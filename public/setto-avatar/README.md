# Setto Avatar

Custom mascot images shown on the dashboard cards when "Setto Avatar" is selected in Settings → Appearance.

## Current files

| File                          | Plugin        |
|-------------------------------|---------------|
| `setto-avatar-difference.png` | Smart Diff    |
| `setto-avatar-search.png`     | Repo Search   |
| `setto-avatar-api.png`        | API Tester    |
| `setto-avatar.png`            | File Editor   |
| `setto-avatar-settings.png`   | Settings      |

## Requirements
- Format: PNG with transparent background recommended
- Suggested size: 512×512 px or larger
- The image is scaled to fill the right side of the card (`object-contain, object-bottom`)

## Switching mascots
Go to **Settings → Appearance → Dashboard mascot** and choose between Panda or Setto Avatar.
